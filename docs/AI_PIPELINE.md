# AI Pipeline

Fessor uses multiple AI providers depending on the task.

Examples include:

- Planning
- Writing
- OCR
- Attachments
- Assistant interactions

The generation process consists of:

1. User prompt
2. Planning
3. Chapter generation
4. Section generation
5. Image generation
6. Quiz creation
7. Flashcard creation
8. Exam creation
9. Database persistence
10. Reader experience

The architecture favors modularity and asynchronous execution.
